共享卷

使用*-volumes-from* 可以把源容器的映射的卷映射到目标容器上。

下面的例子，可以把redis容器的卷映射给Ubuntu容器。*/data仅在Redis容器上存在，但是因为-volumes-from*语句，Ubuntu容器也可以访问到数据。

这样，就可以访问其他容器的卷，而不必关心这个容器是如何跟主机持久化数据的。

只读卷

挂在卷会给容器对目录的全部的读取和写入权限。不过可以使用*:ro*命令来限制为只读权限。

如果容器试图修改数据会发生错误。
